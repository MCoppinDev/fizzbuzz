ABOUT THE CHALLENGE

The FizzBuzz coding challenge was invented by Imran Ghory the objective is given two numbers (usually 3 and 5) we want to write a program that prints onto the console (or webpage in this case) "fizz" if the number is divisible by 3, "buzz" if the number is divisible by 5, or "fizzbuzz" if divisible by both.

Imran had this to say about his reason for this famous interview question:

"After a fair bit of trial and error I’ve come to discover that people who struggle to code don’t just struggle on big problems, or even smallish problems (i.e. write a implementation of a linked list). They struggle with tiny problems. So I set out to develop questions that can identify this kind of developer and came up with a class of questions I call “FizzBuzz Questions” named after a game children often play (or are made to play) in schools in the UK."

-Imran Ghory
